Detailed Description

Представляет узел дерева Хаффмана, используемого для кодирования и декодирования.

Каждый узел хранит символ, его частоту и указатели на левый и правый дочерние узлы. Листовые узлы содержат символы и их частоты, а внутренние узлы объединяют частоты своих дочерних узлов.
