Detailed Description

Реализует кодирование Хаффмана для сжатия и распаковки файлов.

Этот класс предоставляет методы для сжатия и распаковки файлов с использованием алгоритма кодирования Хаффмана. Он создает таблицу частот, строит дерево Хаффмана и генерирует коды Хаффмана для эффективного кодирования. Класс также поддерживает вывод таблицы частот во время распаковки для отладки или анализа.
